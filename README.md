# Accident detection system

## To detect the accidents and send emergency alerts to the nearby hospitals and friends.

## Working 
  - Gyro sensors placed in the helmet detects the sudden impact
  - GPS capures the location
  - GSM sends the alert message

## Components used
  - Microcontroller (ESP-32)
  - Gyro sensors
  - GPS module
  - GSM module

## Objective 
To provide a real time accident detection and alert to save lives of people.
